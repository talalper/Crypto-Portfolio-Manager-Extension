Installation Instructions (Chrome)

To install and test the extension, please follow these steps:

  1. Download the Extension File:

    Click the link below to download the extension file (ZIP file):

    Click here to download the Extension (ZIP)

  2. Extract the ZIP File:

    After downloading, please extract the ZIP file. This will create a new folder containing all the extension files.

  3. Access the Chrome Extensions Page:

    Open your Chrome browser and type the following in the address bar:

    chrome://extensions

    Then press Enter.

  4. Enable Developer Mode:

    On the Extensions page, ensure that the "Developer mode" toggle in the top-right corner is enabled (switched to ON). If it's off, please turn it on.

  5. Load the Extension:

    Click the "Load unpacked" button that appears after enabling Developer mode.

    In the dialog box that opens, navigate to and select the folder you extracted in step 2 (the folder containing the extension files, not the ZIP file itself).

  6. Extension Installed!

    The extension should now appear in your list of extensions. You can click the puzzle piece icon next to the address bar in Chrome and pin the extension for easy access.

Important Note: This extension is loaded in developer mode and requires reloading if the browser is restarted.
